# fetch-github-repositories-service

## Description

it's a service that fetches github repositories and return them back to the
client

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# production mode
npm run build
npm run start

# docker mode
docker-compose up
```

## Test

```bash
# unit tests
npm run test:ci

# coverage
npm run test:coverage
```
